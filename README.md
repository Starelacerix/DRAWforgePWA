# Drawforge PWA

A single-page, offline-first Procreate drawing teacher for beginners.

## New teacher-focused build
- Stronger beginner teaching structure
- Procreate setup steps for each mission
- Procreate tool focus selector
- Teacher focus selector
- Beginner Teacher Lens section
- Common Beginner Traps section
- Self-Check Before Stopping section
- Existing materials, moods, main subject picker, and AI Blueprint Image remain included

## Deploy
Upload all files to the root of a GitHub Pages repo and publish from the root branch folder.

## AI setup
Open Settings in the app, paste your OpenRouter API key, keep the model on `openrouter/free` or set a valid model slug, then use AI Muse or AI Blueprint Image.
